# 滑动窗口

## 无重复字符的最长子串（3）

给定一个字符串 `s` ，请你找出其中不含有重复字符的 **最长 子串** 的长度。

**示例 1:**

```
输入: s = "abcabcbb"
输出: 3 
解释: 因为无重复字符的最长子串是 "abc"，所以其长度为 3。注意 "bca" 和 "cab" 也是正确答案。
```

**示例 2:**

```
输入: s = "bbbbb"
输出: 1
解释: 因为无重复字符的最长子串是 "b"，所以其长度为 1。
```

**示例 3:**

```
输入: s = "pwwkew"
输出: 3
解释: 因为无重复字符的最长子串是 "wke"，所以其长度为 3。
     请注意，你的答案必须是 子串 的长度，"pwke" 是一个子序列，不是子串。
```

```cpp
class Solution {
// 力扣固定要求：解法类为Solution，公有成员函数供平台调用
public:
    // 函数功能：求无重复字符的最长子串长度
    // 形参：s-待处理的输入字符串；返回值：最长无重复子串的长度
    int lengthOfLongestSubstring(string s) {
        // 定义无序哈希集合，存储当前滑动窗口内的字符，实现O(1)快速判重/增/删
        unordered_set<char> occ;
        // 提前计算字符串长度，避免循环内重复调用s.size()，提升效率
        int n = s.size();
        // 右指针rk：初始-1（关键！在字符串左边界左侧，尚未开始移动）
        // 结果变量ans：记录最长无重复子串长度，初始0（处理空字符串边界）
        int rk = -1, ans = 0;
        // 枚举左指针i：作为滑动窗口左边界，慢指针，仅向右移动
        for (int i = 0; i < n; ++i) {
            // 左指针非首次移动时（i>0），执行窗口收缩操作
            if (i != 0) {
                // 左指针右移一格，字符s[i-1]离开窗口，从集合中删除（维护窗口有效性）
                occ.erase(s[i - 1]);
            }
            // 拓展右指针：while循环而非if，要尽可能右移到最长有效位置
            // 条件1：rk+1 < n 保证下一个位置不越界；条件2：!occ.count(...) 保证下一个字符无重复
            while (rk + 1 < n && !occ.count(s[rk + 1])) {
                // 无重复字符入窗，先插入集合标记，再移动指针（顺序不可颠倒）
                occ.insert(s[rk + 1]);
                // 右指针右移一格，更新窗口右边界
                ++rk;
            }
            // 此时[i, rk]是以i为左边界的最长无重复窗口，计算长度并更新最大值
            // 闭区间长度公式：右边界-左边界+1，避免漏算单个字符的情况
            ans = max(ans, rk - i + 1);
        }
        // 遍历完所有左边界，返回最终的最长长度
        return ans;
    }
};
```

复杂度分析

* 时间复杂度：O(N)，其中 N 是字符串的长度。左指针和右指针分别会遍历整个字符串一次。

* 空间复杂度：O(∣Σ∣)，其中 Σ 表示字符集（即字符串中可以出现的字符），∣Σ∣ 表示字符集的大小。在本题中没有明确说明字符集，因此可以默认为所有 ASCII 码在 [0,128) 内的字符，即 ∣Σ∣=128。我们需要用到哈希集合来存储出现过的字符，而字符最多有 ∣Σ∣ 个，因此空间复杂度为 O(∣Σ∣)。



## 找到字符串中所有字母异位词(438)

给定两个字符串 `s` 和 `p`，找到 `s` 中所有 `p` 的 **异位词** 的子串，返回这些子串的起始索引。不考虑答案输出的顺序。

**示例 1:**

```
输入: s = "cbaebabacd", p = "abc"
输出: [0,6]
解释:
起始索引等于 0 的子串是 "cba", 它是 "abc" 的异位词。
起始索引等于 6 的子串是 "bac", 它是 "abc" 的异位词。
```

 **示例 2:**

```
输入: s = "abab", p = "ab"
输出: [0,1,2]
解释:
起始索引等于 0 的子串是 "ab", 它是 "ab" 的异位词。
起始索引等于 1 的子串是 "ba", 它是 "ab" 的异位词。
起始索引等于 2 的子串是 "ab", 它是 "ab" 的异位词。
```

### 滑动窗口

```cpp
class Solution {
public:
    // 函数功能：找到s中所有p的字母异位词的起始索引，返回索引数组
    // 字母异位词：字符相同、个数相同，仅排列顺序不同的字符串
    vector<int> findAnagrams(string s, string p) {
        int sLen = s.size(); // 主串s的长度
        int pLen = p.size(); // 模式串p的长度
        // 边界判断：若s比p短，不可能存在异位词，直接返回空数组
        if (sLen < pLen) {
            return vector<int>();
        }

        vector<int> ans; // 结果数组，存储所有符合条件的起始索引
        vector<int> sCount(26, 0); // 统计s窗口内26个小写字母的频次，初始全0
        vector<int> pCount(26, 0); // 统计p中26个小写字母的频次，初始全0
        // 核心：将字符转成0-25的索引（s[i]-'a'），实现字母到数组下标的映射

        // 初始化窗口：统计s前pLen个字符、p全部字符的频次（窗口长度固定为pLe
        n）
        for (int i = 0; i < pLen; ++i) {
            ++sCount[s[i] - 'a']; // s[i]对应字母频次+1
            ++pCount[p[i] - 'a']; // p[i]对应字母频次+1
        }

        // 首次判断：初始窗口（s[0,pLen-1]）是否是p的异位词
        if (sCount == pCount) {
            ans.emplace_back(0); // 是则记录起始索引0，作用是在 vector 的尾部「直接构造」一个元素（而非拷贝）
        }

        // 滑动窗口：从左到右滑动，每次移动1格，直到窗口右端到达s末尾
        // 循环条件i < sLen - pLen：保证窗口右端i+pLen < sLen，不越界
        for (int i = 0; i < sLen - pLen; ++i) {
            // 窗口左移1格：移除窗口最左侧的字符s[i]，对应频次-1
            --sCount[s[i] - 'a'];
            // 窗口右扩1格：加入窗口右侧新的字符s[i+pLen]，对应频次+1
            ++sCount[s[i + pLen] - 'a'];
            // 频次数组相等 → 窗口内字符是p的异位词，记录当前窗口起始索引（i+1）
            if (sCount == pCount) {
                ans.emplace_back(i + 1);
            }
        }

        return ans; // 返回所有符合条件的起始索引
    }
};
```

复杂度分析

* 时间复杂度：O(m+(n?m)×Σ)，其中 n 为字符串 s 的长度，m 为字符串 p 的长度，Σ 为所有可能的字符数。我们需要 O(m) 来统计字符串 p 中每种字母的数量；需要 O(m) 来初始化滑动窗口；需要判断 n?m+1 个滑动窗口中每种字母的数量是否与字符串 p 中每种字母的数量相同，每次判断需要 O(Σ) 。因为 s 和 p 仅包含小写字母，所以 Σ=26。

* 空间复杂度：O(Σ)。用于存储字符串 p 和滑动窗口中每种字母的数量。

### 优化的滑动窗口

```cpp
class Solution {
public:
    // 函数功能：找到s中所有p的字母异位词的起始索引，返回索引数组
    // 核心优化：单个数组记录频次差 + differ记录差异字母数，避免每次对比26个元素
    vector<int> findAnagrams(string s, string p) {
        int sLen = s.size(), pLen = p.size(); // 主串s、模式串p的长度
        // 边界判断：s比p短，不可能存在异位词，直接返回空数组
        if (sLen < pLen) {
            return vector<int>();
        }

        vector<int> ans; // 结果数组，存储符合条件的起始索引
        vector<int> count(26, 0); // 核心：频次差数组，count[j] = s窗口中j对应字母频次 - p中该字母频次
        // 初始化窗口：统计s前pLen个字符和p所有字符的频次差
        for (int i = 0; i < pLen; ++i) {
            ++count[s[i] - 'a']; // s窗口字符频次+1
            --count[p[i] - 'a']; // p字符频次-1 → 最终count[j]是s窗-p的频次差
        }

        int differ = 0; // 差异数：记录count中「频次差≠0」的字母个数（即s窗和p频次不一致的字母数）
        // 初始化differ：遍历26个字母，统计频次不一致的数量
        for (int j = 0; j < 26; ++j) {
            if (count[j] != 0) {
                ++differ;
            }
        }

        // 差异数为0 → s窗和p所有字母频次完全一致，是异位词，记录起始索引0
        if (differ == 0) {
            ans.emplace_back(0);
        }

        // 滑动固定长度窗口：每次右移1格，直到窗口右端到达s末尾
        // 循环条件i < sLen - pLen：保证窗口右边界i+pLen不越界
        for (int i = 0; i < sLen - pLen; ++i) {
            /********************* 第一步：移除窗口左边界字符s[i] *********************/
            // 先判断移除前的频次差，再更新count数组（顺序不可颠倒！判断的是更新前的状态）
            if (count[s[i] - 'a'] == 1) { 
                // 情况1：移除前，该字母频次差=1（s窗比p多1个）→ 移除后差=0，不一致→一致，differ-1
                --differ;
            } else if (count[s[i] - 'a'] == 0) { 
                // 情况2：移除前，该字母频次差=0（原本一致）→ 移除后差=-1，一致→不一致，differ+1
                ++differ;
            }
            --count[s[i] - 'a']; // 移除左边界字符，s窗频次-1 → 频次差同步-1

            /********************* 第二步：加入窗口右边界新字符s[i+pLen] *********************/
            // 同样先判断加入前的频次差，再更新count数组（顺序不可颠倒）
            if (count[s[i + pLen] - 'a'] == -1) { 
                // 情况1：加入前，该字母频次差=-1（s窗比p少1个）→ 加入后差=0，不一致→一致，differ-1
                --differ;
            } else if (count[s[i + pLen] - 'a'] == 0) { 
                // 情况2：加入前，该字母频次差=0（原本一致）→ 加入后差=1，一致→不一致，differ+1
                ++differ;
            }
            ++count[s[i + pLen] - 'a']; // 加入新字符，s窗频次+1 → 频次差同步+1

            // 差异数为0 → 当前窗口和p频次完全一致，是异位词，记录窗口起始索引i+1
            if (differ == 0) {
                ans.emplace_back(i + 1);
            }
        }

        return ans; // 返回所有符合条件的起始索引
    }
};
```

复杂度分析

* 时间复杂度：O(n+m+Σ)，其中 n 为字符串 s 的长度，m 为字符串 p 的长度，其中Σ 为所有可能的字符数。我们需要 O(m) 来统计字符串 p 中每种字母的数量；需要 O(m) 来初始化滑动窗口；需要 O(Σ) 来初始化 differ；需要 O(n?m) 来滑动窗口并判断窗口内每种字母的数量是否与字符串 p 中每种字母的数量相同，每次判断需要 O(1) 。因为 s 和 p 仅包含小写字母，所以 Σ=26。

* 空间复杂度：O(Σ)。用于存储滑动窗口和字符串 p 中每种字母数量的差。

# <span style="font-size:1.3em; font-weight:bold; font-style:italic; text-decoration:underline;">**ASCII 码差值计算**</span>



 <span style="font-weight:bold; font-size:1.2em;">核心重点：小写英文字母的 ASCII 码规律（代码中最常用）</span>

之前的字母异位词、无重复子串等题目，都限定处理**小写英文字母（a-z）**，而小写字母的 ASCII 码有**连续递增**的关键规律，这是差值计算能生效的核心：

|        字符        |                                                                                      a                                                                                       |   b   |   c   |   d   |  ...  |   y   |   z   |
| :----------------: | :--------------------------------------------------------------------------------------------------------------------------------------------------------------------------: | :---: | :---: | :---: | :---: | :---: | :---: |
| ASCII 码（十进制） |                                                                                      97                                                                                      |  98   |  99   |  100  |  ...  |  121  |  122  |
|      规律总结      | 1. 从 `a` 到 `z` 连续递增，**相邻字母的 ASCII 码差值为 1**；2. `a` 是小写字母的起始，ASCII 码为 **97**；3. `z` 是小写字母的结尾，ASCII 码为 **122**（共 26 个，对应 0-25）。 |       |       |       |       |       |       |

```cpp
// 统计s中字符的频次，count是频次差数组
++count[s[i] - 'a'];
// 统计p中字符的频次
--count[p[i] - 'a'];
```

假设 `s[i] = 'd'`，`p[i] = 'b'`，实际计算过程：

1. `s[i] - 'a' = 'd' - 'a' = 100 - 97 = 3` → 执行 `count[3]++`（统计 d 的频次 + 1）；
2. `p[i] - 'a' = 'b' - 'a' = 98 - 97 = 1` → 执行 `count[1]--`（统计 b 的频次 - 1）。

整个过程编译器会**自动将 char 类型转换为 int 类型**进行运算，无需手动类型转换，C/C++/Java 等语言均支持该特性。

除了小写字母的 `字符 - 'a'`，还有两个最常用的 ASCII 码差值计算，原理完全一致，都是**利用字符 ASCII 码的连续规律做映射**。

1. <span style="font-size:1.2em; font-weight:bold;">大写英文字母：`字符 - 'A'`</span>

大写英文字母（A-Z）的 ASCII 码规律：`A=65`，`B=66`，...，`Z=90`，连续递增，相邻差值为 1。

- 计算：`字符 - 'A'` → 得到 `0 ~ 25` 的整数；
- 例子：`A-'A'=0`，`B-'A'=1`，...，`Z-'A'=25`；
- 应用：处理纯大写字母时，映射为数组 0-25 下标。

2. <span style="font-size:1.3em; font-weight:bold;">数字字符：`字符 - '0'`</span>

数字字符（'0'-'9'）的 ASCII 码规律：`'0'=48`，`'1'=49`，...，`'9'=57`，连续递增，相邻差值为 1。

- 计算：`数字字符 - '0'` → 得到**对应的整型数字（0-9）**；
- 例子：`'0'-'0'=0`，`'5'-'0'=5`，`'9'-'0'=9`；
- 应用：将输入的数字字符（如字符串中的 '5'）转换为可计算的整型数字，是处理数字字符串的核心技巧。

```cpp
string numStr = "123";
int num = 0;
for (char c : numStr) {
    num = num * 10 + (c - '0'); // 每次将字符转成数字，累计算
}
// 结果：num = 123
```
