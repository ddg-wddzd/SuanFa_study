

## 按位运算

#### 核心按位运算符（附规则 + 示例）

| 运算符                                                       | 名称     | 运算规则                                                     | 示例（以 5 (0101) 和 3 (0011) 为例） |      |      |
| :----------------------------------------------------------- | :------- | :----------------------------------------------------------- | :----------------------------------- | :--- | :--- |
| `&`                                                          | 按位与   | 对应位都为 1 则结果为 1，否则为 0                            | `5 & 3 = 1`（0001）                  |      |      |
| `      | 按位或   | 对应位只要有 1 则结果为 1，否则为 0                          | 3  \`5= 7（0111） |          |                                                              |                                      |      |      |
| `^`                                                          | 按位异或 | 对应位不同为 1，相同为 0（特性：`x^x=0`、`x^0=x`）           | `5 ^ 3 = 6`（0110）                  |      |      |
| `~`                                                          | 按位取反 | 所有位取反（0→1，1→0，注意 C++ 补码机制：`~5 = -6`）         | `~5 = -6`                            |      |      |
| `<<`                                                         | 左移     | 所有位左移 n 位，右侧补 0（无溢出时等价于`×2?`）             | `5 << 1 = 10`（1010）                |      |      |
| `>>`                                                         | 右移     | 所有位右移 n 位，左侧补符号位（正数补 0，负数补 1，等价于`÷2?`取整） | `5 >> 1 = 2`（0010）                 |      |      |

#### 2. 按位运算代码示例

```cpp
#include <iostream>
using namespace std;
int main() {
    int a = 5, b = 3;
    // 按位与：判断奇偶（x&1=1则奇，=0则偶）
    cout << "5 & 3 = " << (a & b) << endl;   // 输出1
    // 按位或：置指定位为1
    cout << "5 | 3 = " << (a | b) << endl;   // 输出7
    // 按位异或：交换两个数（无需临时变量）
    a ^= b; b ^= a; a ^= b;
    cout << "交换后a=" << a << ", b=" << b << endl; // a=3, b=5
    // 左移：快速乘2
    cout << "3 << 2 = " << (3 << 2) << endl; // 输出12（3×4）
    return 0;
}
```

## 数组与高精度运算

数组是 C++ 中**连续存储相同类型数据**的集合，下标从 0 开始。由于 C++ 基本类型（`int`/`long long`）存储范围有限（`long long`最大约 9e18），高精度运算必须依赖数组存储超大数的每一位。

```cpp
#include <iostream>
using namespace std;
int main() {
    // 定义能存1000位的数组，初始化为0（高精度常用长度）
    int big_num[1000] = {0};
    // 逆序存储数字12345（下标0存个位，方便进位处理）
    big_num[0] = 5; // 个位
    big_num[1] = 4; // 十位
    big_num[2] = 3; // 百位
    big_num[3] = 2; // 千位
    big_num[4] = 1; // 万位
    // 输出原数（从高位到低位遍历）
    cout << "存储的数字：";
    for (int i = 4; i >= 0; i--) {
        cout << big_num[i];
    } // 输出：12345
    return 0;
}
```

#### 2. 数组在高精度中的关键作用

- 拆分超大数：将 100 位的数拆成单个数字，存到数组的不同下标；
- 逆序存储：下标 0 对应个位，便于模拟手工计算的 “进位 / 借位”；
- 连续访问：数组的连续内存特性，可高效遍历每一位进行运算。

### 高精度运算（数组 + 模拟手工计算）

高精度运算的核心是**用数组存储超大数的每一位，模拟手工加减乘除的过程**，解决基本类型存储范围不足的问题。以下以最基础的**高精度加法**为例讲解。

### <span style="font-weight:bold; font-style:italic; text-decoration:underline; font-size:1.1em;">高精度加法</span>

```cpp
#include <iostream>
#include <string>
#include <algorithm> // max函数
using namespace std;
const int MAX_LEN = 1005; // 支持最大1000位数字
int main() {
    // 用字符串接收超大数（避免输入溢出）
    string s1, s2;
    cout << "输入第一个大数：";
    cin >> s1;
    cout << "输入第二个大数：";
    cin >> s2;
    // 定义数组存储逆序后的数字，初始化为0
    int a[MAX_LEN] = {0}, b[MAX_LEN] = {0}, res[MAX_LEN] = {0};
    int len1 = s1.size(), len2 = s2.size();
    // 逆序存储：字符串最后一位（个位）存到数组下标0
    for (int i = 0; i < len1; i++) {
        a[i] = s1[len1 - 1 - i] - '0'; // 字符转数字（'0'的ASCII码为48）
    }
    for (int i = 0; i < len2; i++) {
        b[i] = s2[len2 - 1 - i] - '0';
    }
    // 逐位相加+处理进位
    int carry = 0; // 进位标志，初始为0
    int max_len = max(len1, len2); // 取较长数的长度作为循环上限
    for (int i = 0; i < max_len; i++) {
        int sum = a[i] + b[i] + carry;
        res[i] = sum % 10; // 当前位结果
        carry = sum / 10;  // 更新进位
    }
    // 最后还有进位，补到结果数组
    if (carry != 0) {
        res[max_len] = carry;
        max_len++;
    }
    // 逆序输出结果（从高位到个位）
    cout << "相加结果：";
    for (int i = max_len - 1; i >= 0; i--) {
        cout << res[i];
    }
    cout << endl;
    return 0;
}
```

### <span style="font-weight:bold; font-style:italic; text-decoration:underline; font-size:1.1em;">高精度减法</span>

#### 核心逻辑

1. **大小判断**：若被减数 < 减数，交换两数并标记结果为负；若相等，结果直接为 0；
2. **逆序存储**：同加法，字符串逆序转为数组（下标 0 存个位）；
3. **逐位相减**：`当前位 = 被减数[i] - 减数[i] - 借位`；若当前位 < 0，向高位借 1（借位 = 1，当前位 + 10），否则借位 = 0；
4. **去前导零**：结果数组高位可能有多个 0（如 1000-999=1），需跳过这些 0 再输出；
5. **输出结果**：若有负号先输出，再逆序打印结果数组。

```cpp
#include <iostream>
#include <string>
#include <algorithm>
using namespace std;
const int MAX_LEN = 1005;
// 辅助函数：比较两个字符串大数的大小（a >= b 返回true）
bool compare(string a, string b) {
    if (a.size() != b.size()) return a.size() > b.size();
    for (int i = 0; i < a.size(); i++) {
        if (a[i] != b[i]) return a[i] > b[i];
    }
    return true; // 两数相等
}
int main() {
    string s1, s2;
    cout << "输入被减数（大数）：";
    cin >> s1;
    cout << "输入减数（大数）：";
    cin >> s2;
    int a[MAX_LEN] = {0}, b[MAX_LEN] = {0}, res[MAX_LEN] = {0};
    int len1 = s1.size(), len2 = s2.size();
    bool is_negative = false; // 标记结果是否为负
    // 确保被减数 >= 减数，否则交换并标记负号
    if (!compare(s1, s2)) {
        swap(s1, s2);
        swap(len1, len2);
        is_negative = true;
    }
    // 逆序存储到数组（个位存下标0）
    for (int i = 0; i < len1; i++) a[i] = s1[len1-1-i] - '0';
    for (int i = 0; i < len2; i++) b[i] = s2[len2-1-i] - '0';
    // 逐位相减+处理借位
    int borrow = 0; // 借位标志
    for (int i = 0; i < len1; i++) {
        int sub = a[i] - b[i] - borrow;
        if (sub < 0) { // 需要借位
            sub += 10;
            borrow = 1;
        } else {
            borrow = 0;
        }
        res[i] = sub;
    }
    // 去掉前导零（找到最高位）
    int max_len = len1;
    while (max_len > 1 && res[max_len-1] == 0) max_len--;
    // 输出结果
    cout << "相减结果：";
    if (is_negative) cout << "-";
    for (int i = max_len-1; i >= 0; i--) cout << res[i];
    cout << endl;
    return 0;
}
```

### <span style="font-weight:bold; font-style:italic; text-decoration:underline; font-size:1.1em;">高精度乘法</span>

####  核心逻辑

「位级规律」：

- 123 的**个位（3，对应 10?）** × 45 的**个位（5，对应 10?）** → 结果属于 10?位（个位）；
- 123 的**个位（3，对应 10?）** × 45 的**十位（4，对应 10?）** → 结果属于 10? 位（十位）；
- 123 的**十位（2，对应 10?）** × 45 的**个位（5，对应 10?）** → 结果也属于 10? 位（十位）；
- 总结：被乘数的第`i`位（对应 10^?^） × 乘数的第`j`位（对应 10^?^） → 结果属于 10^(i+j)^ 位。



- 为了匹配手工计算顺序，我们把大数**逆序存储**（个位存在数组下标 0 的位置），具体步骤如下：
  1. **输入接收**：用字符串接收两个大数（直接输数字会溢出）；
  2. **逆序转数组**：把字符串拆成单个数字，逆序存入数组（如 "123"→[3,2,1]）；
  3. **错位累加乘积**：遍历两个数的每一位，将`a[i]×b[j]`累加到结果数组的`res[i+j]`位置（核心规律）；
  4. **统一处理进位**：遍历结果数组，把每位的数值拆为「当前位（%10）」和「进位（/10）」；
  5. **去前导零**：结果数组高位可能有多余的 0（如 1×1=1，数组可能存 [1,0,0]），需跳过；
  6. **逆序输出**：把结果数组逆序打印，得到最终结果。

```cpp
#include <iostream>
#include <string>
using namespace std;
// 乘法结果最大长度 = 两个数长度之和（如100位×100位=200位），设足够大的数组
const int MAX_LEN = 2005;
int main() {
    // 步骤1：接收两个超大数（字符串避免溢出）
    string num1, num2;
    cout << "输入第一个大数：";
    cin >> num1;
    cout << "输入第二个大数：";
    cin >> num2;
    // 步骤2：定义数组并初始化（全0）
    int a[MAX_LEN] = {0};  // 存储第一个数的逆序（个位→下标0）
    int b[MAX_LEN] = {0};  // 存储第二个数的逆序
    int res[MAX_LEN] = {0};// 存储乘积结果（先累加，后处理进位）
    // 步骤3：字符串逆序转数组（字符→数字）
    int len1 = num1.size(); // 第一个数的长度
    int len2 = num2.size(); // 第二个数的长度
    for (int i = 0; i < len1; i++) {
        // num1[len1-1-i]：取字符串倒数第i+1位（如"123"的i=0→'3'）
        // -'0'：字符转数字（'3'-'0'=3）
        a[i] = num1[len1 - 1 - i] - '0';
    }
    for (int i = 0; i < len2; i++) {
        b[i] = num2[len2 - 1 - i] - '0';
    }
    // 步骤4：核心——错位累加所有位的乘积
    for (int i = 0; i < len1; i++) { // 遍历第一个数的每一位（对应10^i）
        for (int j = 0; j < len2; j++) { // 遍历第二个数的每一位（对应10^j）
            // a[i]×b[j]属于10^(i+j)位，直接累加到res[i+j]
            res[i + j] += a[i] * b[j];
            // 先不处理进位，全部累加完再统一处理（效率更高）
        }
    }
    // 步骤5：统一处理进位
    for (int k = 0; k < len1 + len2; k++) { // 结果最大长度=len1+len2
        if (res[k] >= 10) { // 有进位
            res[k + 1] += res[k] / 10; // 进位值加到下一位
            res[k] = res[k] % 10;      // 保留当前位的个位数
        }
    }
    // 步骤6：去除前导零（找到最高位）
    int result_len = len1 + len2; // 初始结果长度
    // 从后往前找第一个非0数，就是最高位
    while (result_len > 1 && res[result_len - 1] == 0) {
        result_len--;
    }
    // 步骤7：逆序输出结果（res是逆序存储的）
    cout << "乘法结果：";
    for (int i = result_len - 1; i >= 0; i--) {
        cout << res[i];
    }
    cout << endl;
    return 0;
}
```

### <span style="font-weight:bold; font-style:italic; text-decoration:underline; font-size:1.1em;">高精度除法</span>

和加法 / 乘法不同，除法需要**正序处理**（从高位到低位），因此无需逆序存储，直接按字符串原顺序遍历即可。具体步骤：

1. **输入接收**：用字符串存被除数（超大数，避免溢出），用整数存除数（普通 int/long long 即可）；
2. **防除零校验**：除数不能为 0，提前判断并报错；
3. **初始化变量**：余数（初始 0）、商的存储容器（用 vector 更灵活，避免固定数组长度限制）；
4. **逐位计算**：遍历被除数的每一位，按核心公式计算商和余数；
5. **去前导零**：商的高位可能有多余的 0（如 123÷456 → 商的前几位都是 0），需跳过；
6. **输出结果**：打印商和余数（商为 0 时需特殊处理）。

```cpp
#include <iostream>
#include <string>
#include <vector> // 用vector存储商，无需提前定长，新手更易理解
using namespace std;
int main() {
    // 步骤1：接收输入（被除数是大数，用字符串；除数是普通整数）
    string dividend_str; // 被除数（如"12345678901234567890"）
    long long divisor;   // 除数（用long long避免除数溢出，如1e9）
    cout << "输入被除数（超大数）：";
    cin >> dividend_str;
    cout << "输入除数（非0整数）：";
    cin >> divisor;
    // 步骤2：防除零错误（必须先判断）
    if (divisor == 0) {
        cout << "错误：除数不能为0！" << endl;
        return 1; // 异常退出
    }
    // 步骤3：初始化变量
    vector<int> quotient; // 存储商的每一位（正序，高位→低位）
    long long remainder = 0; // 余数，初始为0（用long long避免余数溢出）
    // 步骤4：核心——逐位计算商和余数（正序遍历被除数）
    for (char c : dividend_str) { // 遍历被除数的每一位（如"1234"→'1'→'2'→'3'→'4'）
        int digit = c - '0'; // 字符转数字（'1'-'0'=1）
        remainder = remainder * 10 + digit; // 余数传递到当前位
        quotient.push_back(remainder / divisor); // 试商，存入商的数组
        remainder = remainder % divisor; // 更新余数，用于下一位计算
    }
    // 步骤5：去除商的前导零（关键！）
    int start_idx = 0; // 商的有效起始下标
    while (start_idx < quotient.size() && quotient[start_idx] == 0) {
        start_idx++; // 跳过所有前导零
    }
    // 步骤6：输出结果（分情况处理）
    cout << "除法结果：\n";
    // 情况1：商为0（如123÷456，所有位都是0）
    if (start_idx == quotient.size()) {
        cout << "商：0\n";
    }
    // 情况2：商非0，输出有效位
    else {
        cout << "商：";
        for (int i = start_idx; i < quotient.size(); i++) {
            cout << quotient[i];
        }
        cout << "\n";
    }
    // 输出余数（无论商是否为0，余数都要输出）
    cout << "余数：" << remainder << endl;
    return 0;
}
```

#### 关键细节解释（新手必看）

1. **为什么用 vector 存商？**

   被除数的长度不确定（可能 100 位、1000 位），vector 无需提前指定长度，能动态添加商的每一位，比固定数组更灵活，避免浪费空间或长度不足。

   

2. **余数为什么用 long long？**

   假设除数是`1e9`（long long 范围），余数 = 上一轮余数 ×10 + 当前位（最大 9），若用 int 存储，`1e9×10=1e10`会超出 int 范围（int 最大约 2e9），导致溢出，long long 能避免这个问题。

   

3. **去前导零的必要性？**

   比如计算`123 ÷ 456`时，逐位计算的商是`[0,0,0]`，若直接输出会显示 “000”，去前导零后才会正确显示 “0”；再比如`1000 ÷ 2`，商的初始存储是`[0,5,0,0]`，去前导零后显示 “500”。
